Hermes Atlas serves as the central community map for the Hermes Agent ecosystem. It tracks every available tool, skill, and integration created by Nous Research and the developer community. This resource makes it simple to find the right components to improve your AI agents.

## 📍 About the map

Modern AI agents require specific tools to perform tasks effectively. The Hermes ecosystem grows each day with new capabilities. This directory organizes those capabilities into a searchable format. You can use this map to identify which skills your agent needs to complete complex projects. 

The project focuses on three main categories:

- Tools: Software interfaces that allow agents to interact with external databases or websites.
- Skills: Pre-trained behaviors that give agents the ability to reason through specific problems. 
- Integrations: Connectors that link your agent to other platforms like messaging services or project management boards.

## ⚙️ Using the interface

The application loads a visual map upon startup. You will see various nodes representing different agent tools. 

### Searching the map

The search bar resides at the top of the window. Type the name of a task or a tool to filter the results. The map updates in real time to show only the items related to your search term. 

### Viewing tool details

Click on any node to open the detail panel. This panel displays information about the specific tool or skill, including:

- A description of the tool function.
- The requirements for implementing the tool.
- Links to documentation or support forums.
- Tags that indicate the category and compatibility of the tool.

### Sorting and filtering

Use the menu on the left side of the screen to group tools by category. You can toggle visibility for skills or integrations to clear the screen and focus on the information you need. 

## 🛠️ Frequently asked questions

### Does this map track live status?

Yes. The map pulls data from live sources to ensure you see the current state of the ecosystem. It highlights tools that are active and compatible with the most recent versions of Hermes Agent.

### Can I add my own tools?

The community manages this map. If you create a new integration or skill for Hermes, you can submit it to the repository. The process involves filling out a simple form within the application. Once approved, your tool appears on the map for others to find.

### Does the software send data back?

The application collects minimal usage statistics to improve map navigation. It does not track personal files or private agent data. You can disable these analytics in the settings menu.

### Why do some nodes look greyed out?

Grey nodes represent tools that are currently deprecated or under review. These tools may no longer work with the latest version of the Hermes Agent software. We keep them on the map for historical reference.
